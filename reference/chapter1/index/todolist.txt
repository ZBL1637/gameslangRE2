## 三、全局 UI 体系 & 复古 RPG 主题

* [x] **设计全局主题变量（可以写成 `theme.ts` 或 SCSS 变量）**
  * [x] 主色：深蓝/墨绿；副色：金色描边；背景：复古土黄/草绿；高亮：电子蓝。
  * [x] 字体：像素风英文字体 + 一款可读性好的中文字体（通过 CSS 做轻描边、阴影假装像素感）。
  * [x] 定义统一阴影和边框风格：RPG 菜单框、金色描边、半透明面板。
* [x] **实现基础 UI 组件库**
  * [x] `<GameFrame>`：16:9 的「游戏画面框」，负责 letterbox 黑边效果。
  * [x] `<Panel>`：深色 + 金边的内容面板（用在菜单、对话框、章节卡片等）。
  * [x] `<Button>`：三种状态（默认、hover、禁用），支持左侧像素小三角选中态。
  * [x] `<HUDBar>`：顶部/底部状态栏（用在等级、EXP、世界统计、提示语）。
  * [x] `<DialogBox>`：RPG 对话框，支持头像、文字打字机动画。
  * [x] `<Tooltip>`：术语 hover 时弹出的解释框（简要解释 + 日常语言翻译）。
  * [x] `<Icon>` 集合：锁、勾、星星、EXP 水晶、小旗、键盘提示箭头等。
* [x] **全局布局**
  * [x] 在根组件中放置 `GameFrame`，内部通过路由切换不同「画面」。
  * [x] 统一加载背景：暗房间 + 中间老电视机/显示器效果。
